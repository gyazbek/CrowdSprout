Crowdsourced Seed Generator


About

CrowdSprout is a crowdsourced random number generator. The random actions that users take on this website are converted into numbers, which becomes seeds. This number is then uniquely shared with other users. When a user receives a seed, that piece of data is used to further generate the random number you see.

Every password generation occurs on the client side, so your end result is not shared with anyone. The server doesn't store seed data outside of a temporary queue. if there isn't enough activity, the server might generate a random seed based on as-random-as-possible data, such as cpu usage or temperature, if available.
